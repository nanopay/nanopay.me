# NanoPay.me API Routes

This directory contains the REST API routes for the NanoPay.me service.

It expose the necessary functionalities required for seamless integration with the NanoPay.me platform by merchants.

- Create invoices
- Create webhooks
- Access invoice payments

## Important Note [Contributors]

These routes are designed solely to facilitate the integration of the NanoPay.me API by merchants and should not be used internally or within NanoPay.me frontend.
