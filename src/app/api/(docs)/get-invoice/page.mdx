## Get Invoice

Retrieve details of a specific invoice.

```http
GET /invoices/{id}
```

#### Parameters

| Name | Type   | In   | Description                          |
| ---- | ------ | ---- | ------------------------------------ |
| `id` | string | path | The unique identifier of the invoice |
