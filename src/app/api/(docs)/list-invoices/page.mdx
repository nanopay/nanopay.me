# List All Invoices

Retrieve a list of all invoices.

```http
GET /invoices?limit=10&offset=0&order=desc
```

#### Query Parameters

| Name     | Type    | Required | Description                                                                  |
| -------- | ------- | -------- | ---------------------------------------------------------------------------- |
| `limit`  | integer | Optional | The maximum number of invoices to return. Range: `1` to `20` (default: `10`) |
| `offset` | integer | Optional | The number of invoices to skip (for pagination). (default: `0`)              |
| `order`  | string  | Optional | Order of `created_at`. Allowed values: `asc` or `desc` (default: `desc`)     |
